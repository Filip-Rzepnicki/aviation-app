# Airplanes API

Prosta aplikacja REST API w Spring Boot do zarządzania danymi o samolotach (Aircraft).
Udostępnia operacje CRUD, wyszukiwanie oraz podstawową autoryzację (Basic Auth) z rolami USER/ADMIN.

## Technologie

- Java
- Spring Boot
- Spring Web (REST)
- Spring Data JPA
- Spring Security (Basic Auth)
